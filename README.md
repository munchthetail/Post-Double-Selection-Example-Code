# Post-Double-Selection-Example-Code
This repository provides example code in R and Stata for the use cases of PDS-lasso utilizing publically available datasets.

Downloand the entire repository, then open any of the Example.do files. Set the directory to the path of the saved repository folder.

The program will automatically install neccssary dependencies.

Datasets collected from Rdatasets.

Arel-Bundock, Vincent. Rdatasets: A Collection of Datasets Originally Distributed in Various R Packages. GitHub repository. 2025. https://github.com/vincentarelbundock/Rdatasets
